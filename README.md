# Gamut Installer
=======================================================================


## 1. Download Gamut Installer (372MB - Contains MongoDB, Wildfly Server and sample vcf data) 

    https://drive.google.com/uc?id=17Fek9jop8S2oT-54pxGqrTgG8FFobFcT&export=download

    
## 2. Unzip  
   
    unzip gamut.zip  
    cd gamut/
   
## 3. Run Gamut   
    
    ./gamut.sh

## 4. Open into the Browser
    
    http://localhost:8080/
